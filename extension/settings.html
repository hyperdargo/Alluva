The Native Messaging Host is a lightweight Node.js script that allows this extension to save files directly to any path on your drive and launch VLC automatically.
To install and register the native host on your PC:
1. Open PowerShell or command line. 2. Navigate to your Stream Vault workspace directory: cd "O:\website\native-host" 3. Run the host registration script: .\install_host.bat
